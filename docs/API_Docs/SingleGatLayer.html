SingleGatLayer¶

Single graph attention network for predicting crystal properties.

Note

Some abbreviations used in GATLayer class:

Abbreviations

Full name

dist

distance matrix

feat

features

ft

features

src

source node

dst

destination node

e

e_i_j: refer to: https://arxiv.org/abs/1710.10903

a

alpha_i_j: refer to: https://arxiv.org/abs/1710.10903

att

attention mechanism

act

activation function
class GATLayer¶
__init__(self, num_out, num_heads, bias=False, negative_slope=0.2, activation=None, batch_normalization=False)¶
Parameters:

  • num_out (int) – Depth of node representation in the output of each head of this GAT layer.

  • num_heads (int) – Number of attention heads.

  • bias (bool) – Whether the dense layer uses a bias vector.

  • negative_slope (float) – Negative slope coefficient of the leakeyrelu activation function.

  • activation (str/tf.keras.activations) – activation function to use

  • batch_normalization (bool) – if True, append a tf.keras.layers.BatchNormalization() layer to this AGAT layer.

build(self, input_shape)¶

The build method is thoroughly introduced by TensorFlow. https://www.tensorflow.org/guide/keras/custom_layers_and_models In short, this method creates variables at the first __call__ of this class.

call(self, feat, dist, graph)¶

Forward this GAT layer.

Parameters:

  • feat (tf.Tensor) – input features of all nodes (atoms).

  • dist (tf.Tensor) – distance matrix.

  • graph (DGL.graph) – A graph built with DGL.

Returns:

dst: output features of all nodes.
