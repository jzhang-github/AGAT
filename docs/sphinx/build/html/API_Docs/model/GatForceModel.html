GatForceModel¶

Construct a force model.

Note

You can also use this model to train and predict bond related properties. You need to store the labels on graph edges if you want to do so.

Note

This model has multiple attention heads.

class GAT(tf_model)¶
__init__(self, num_gat_out_list, num_readout_out_list=[1], head_list_force=['div'], embed_activation='LeakyReLU', readout_activation='LeakyReLU', bias=True, negative_slope=0.2, batch_normalization=False, tail_readout_no_act=1)¶
Parameters:

  • num_gat_out_list (list) – stack multiple AGAT layers. A list of numbers that contains the representation dimension of each GAT layer.

  • num_readout_out_list (list) – stack multiple neural network (densely connected) layers. A list of numbers that contains the representation dimension of each readout layer.

  • head_list_force (list) – A list contains the attention mechanisms of each head for the energy prediction.

  • embed_activation (str) – TensorFlow activation function for embedding the inputs.

  • readout_activation (str) – TensorFlow activation function for the readout layers.

  • bias (bool) – bias term of dense layers.

  • negative_slope (float) – Negative slope of LeakyReLu function.

  • batch_normalization (bool) – if True, append batch_normalization to this model.
