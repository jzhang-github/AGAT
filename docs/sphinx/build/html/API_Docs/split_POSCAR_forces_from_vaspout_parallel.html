split_output(in_path_list, out_path, working_dir, process_index, mask_similar_frames=True, energy_stride=0.05)¶

Read VASP outputs and seperate the outputs into a dataset.

Note

You may find information at AGAT_CATA_example.

Parameters:

  • in_path_list (list) – A list of absolute paths where OUTCAR and XDATCAR files exist.

  • out_path (str) – Absolute path where the collected data to save.

  • working_dir (str) – The working directory.

  • process_index (int/str) – index to differentiate output file names.

  • bool (mask_similar_frames) –

    if True, mask frames in VASP OUTCAR with similar energies.

    Note

    If mask_similar_frames=True, the energy difference between collected frames is larger than energy_stride defined below.

  • energy_stride (float) – specify energy increaments if you need to mask similar frames.

Main process to seperate VASP outputs in parallel.
